# 🚀 Personal Portfolio

This is my personal developer portfolio built with **React, Vite, and Tailwind CSS**, showcasing my projects, skills, and experience as a Computer Science and Engineering student and aspiring Machine Learning Engineer.

🌐 Live Site: https://portfolio-7b922.web.app/

---

## ✨ Features

- ⚡ Fast and responsive UI (Vite + React)
- 🎨 Modern design with Tailwind CSS
- 📱 Fully mobile responsive
- 🧑 About section with introduction
- 💼 Projects showcase with live links and GitHub repos
- 🛠 Skills section grouped by categories
- 📄 Downloadable resume
- 📬 Contact section
- 🌙 Clean and minimal UI

---

## 🛠 Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Firebase Hosting
